## 🌐 公开测试API

### 推荐API列表

| API | 稳定性 | Swagger配置 | 特点 |
|-----|--------|------------|------|
| **JSONPlaceholder** | ⭐⭐⭐⭐⭐ | ✅ 已提供 | 最稳定，完全免费 |
| **ReqRes** | ⭐⭐⭐⭐ | ❌ 需创建 | 数据结构真实 |
| **Petstore** | ⭐⭐⭐ | ❌ 需下载 | Swagger官方示例 |

### 使用方式

#### JSONPlaceholder（推荐）
```bash
# 已提供配置文件，直接运行
python src/pipeline.py swagger_jsonplaceholder.json
```

#### Petstore
```bash
# 下载Swagger文档
curl https://petstore.swagger.io/v2/swagger.json -o swagger_petstore.json

# 运行测试
python src/pipeline.py swagger_petstore.json
```

#### ReqRes
```bash
# 1. 创建配置文件 swagger_reqres.json（参考公开测试API.md）
# 2. 运行测试
python src/pipeline.py swagger_reqres.json
```

```bash

```

```bash

```

## 💡 快速命令

```bash
# 安装依赖
pip install pytest requests

# 运行测试（JSONPlaceholder）
python src/pipeline.py swagger_jsonplaceholder.json

# 查看报告
cat reports/api_test_report_*.txt

# 手动运行测试
pytest tests/ -v

# 查看生成的测试文件
ls -la tests/
```
